## Technical Implementation

In-Depth Analysis of XOR Encryption and Decryption
The core encryption mechanism of this tool is based on the bitwise XOR (Exclusive OR) operation. XOR is a fundamental yet powerful binary logic operation with a simple rule: the output is 0 if the two input bits are identical, and 1 if they differ. Mathematically, it possesses a remarkable "self-inverse" property: if A ^ B = C, then C ^ B = A.

### 1. Encryption Process:

When you input text, the program first converts it into a UTF-8 character stream. It then generates a random alphanumeric string to serve as the "key." During the encryption loop, each byte of the plaintext undergoes a bitwise XOR operation with the corresponding character code from the key. Since the key is typically shorter than the text, the program cycles through the key using a modulo operation (i % key.length). This process scrambles the original readable characters into seemingly random hexadecimal values, making it impossible for anyone without the key to decipher the original text.

### 2. Data Encapsulation:

To ensure stable transmission within URLs, the program converts the XORed hexadecimal result into URL-safe Base64 encoding (replacing `+` with `-` and `/` with `_`). This guarantees that the ciphertext—even if it contains special binary bits—can be transmitted losslessly as a URL parameter across various browsers and social platforms.

### 3. Decryption and Security:

Decryption is essentially the reverse of the encryption process. After retrieving the key from the URL, the browser decodes the Base64 ciphertext and performs the exact same XOR operation with the key. Thanks to the properties of XOR, this step cancels out the initial scrambling, instantly restoring the original UTF-8 byte stream.

The advantage of this approach lies in its superior privacy: all computations (encoding, XORing, and encapsulation) take place entirely within your local browser cache. Since no backend server is involved, the key and ciphertext exist only within the shared URL link, achieving true "zero-trace" privacy protection.
